# 知了

知了 PWA 客户端, 支持 Web 浏览器.

## 关联项目

- cicada_server: 知了服务器端, 即将开源
- [cicada_electron](https://github.com/mebtte/cicada_electron): 知了 Electron 客户端, 支持 macOS/Windows

## 开发/构建

### 开发

```bash
npm install
npm run dev # localhost:8000
```

### 构建

```bash
npm run build # 构建后资源位于 build 目录
```
